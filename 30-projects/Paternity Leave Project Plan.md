## Feasibility Assessment

### Overall Verdict: Ambitious but Achievable (75%)

The project list is large for a first-time paternity leave. Sleep deprivation and unpredictable baby schedule will compress daily output significantly. That said, the projects are **sequenced logically**, and the timeline has enough buffer to absorb 3–4 slow days.

### Time Estimates by Project

| Project | Est. Days | Confidence |
|---|---|---|
| Prework / Measurements | 1–2 | High |
| Move Sleeper Couch | 1 | High |
| Move Nursery Crib/Bed | 0.5–1 | High |
| Garage Demo (workbench + shoe storage) | 2–3 | High |
| Garage Storage Racks Build | 3–4 | Medium |
| TV Mount (pro install) | 0.5 | High |
| IKEA Holmerud Purchase + Assembly | 1.5–2 | High |
| Under Stairs Storage Build | 6–9 | Low–Medium |
| Console Table Build | 3–5 | Medium |
| Mounted Shower Head Install | 0.5–1 | High |
| Replace Door Handles (6 doors) | 0.5–1 | High |
| **Total (without vacation)** | **20–31 days** | |

### Risks

- **Under Stairs Storage** is the highest-risk project. Angled cuts, custom fit, and multiple components (frame, shelves, door/panel) can balloon in time. Budget 8 days and treat any finishing by Day 29 as a win.
- **Console Table** depends on your woodworking comfort. If routing or joinery is involved, add 1–2 days.
- **Newborn variability** — some weeks will be rough. Build your schedule around phases, not rigid days, so slippage doesn't derail everything.
- **Material delays** — order anything with lead time (hardware, specialty lumber, fixtures) during Day 2 prework.

### What to Protect If Time Compresses

**Must Do:** Move furniture, IKEA assembly, Garage demo, Mounted shower head  
**High Priority:** Garage storage racks, TV mount  
**Deprioritize if needed:** Under stairs doors/detail work, console table finishing

### PHASE 1 — Prework & Planning (Days 1–2)

**Day 1 — Measurements**

- [ ] Photograph both spaces from multiple angles
- [ ] Sketch rough design concepts for both builds
- [ ] **Door backset** — remove hardware from one interior door, measure from door edge to center of hole; record result (2-3/8" or 2-3/4") so you can order the right handles on Day 2

**Day 2 — Planning & Procurement**
- [ ] Build detailed shopping/materials list for every project
- [ ] Research lumber prices, plywood sheets, hardware

- [ ] Call/text help for Day 3 couch move

### PHASE 6 — Under Stairs Storage Build (Days 21–29)

*This is the longest and most complex project. Give it room to breathe.*

**Day 21 — Finalize Design**
- [ ] Review measurements and photos from Day 1
- [ ] Decide on configuration: open shelves vs. cabinet vs. hybrid
- [ ] Create cut list (all pieces by dimension and quantity)
- [ ] Decide on door/access panel style (full door, multiple panels, angled door)

**Day 22 — Materials Run**
- [ ] Buy: plywood (3/4"), 1×4 or 2×4 framing, hinges, handles, wood filler, primer, paint
- [ ] Buy: jigsaw or circular saw blades for angled cuts if needed
- [ ] Pick up any specialty items (piano hinge for angled door, magnetic catches)

**Day 23 — Build Outer Frame / Carcass**
- [ ] Build the outer box/frame that fits inside the stair cavity
- [ ] Account for angled top edge (stair angle) — use your measurements
- [ ] Dry-fit in space before fastening everything permanently

**Day 24 — Test Fit + Trim Angles**
- [ ] Slide carcass into stair opening
- [ ] Mark and trim any areas that don't fit flush
- [ ] Shim level if floor is uneven
- [ ] Confirm structural connection to wall/framing

**Day 25 — Interior Shelving**
- [ ] Cut and install fixed shelves inside carcass
- [ ] Account for decreasing height as shelves approach the angled stair end
- [ ] Add shelf pins or dado grooves for adjustable shelves if desired

**Day 26 — Door / Access Panel**
- [ ] Build door panel(s) from plywood or MDF
- [ ] Cut top edge at stair angle if needed
- [ ] Hang with hinges, test swing clearance
- [ ] Install handle and magnetic/ball catch

**Day 27 — Sand, Prime, Paint**
- [ ] Fill nail holes and gaps with wood filler, let dry
- [ ] Sand all surfaces (120 → 220 grit)
- [ ] Prime all surfaces (especially cut edges of plywood)
- [ ] First coat of paint — let dry fully

**Day 28 — Second Coat + Install**
- [ ] Second coat of paint
- [ ] Caulk any gaps around the frame perimeter
- [ ] Install any trim or molding to conceal edges
- [ ] Install baseboard or shoe mold at floor line

**Day 29 — Load + Touch-ups**
- [ ] Load items into storage
- [ ] Touch-up any paint nicks from install
- [ ] Step back and appreciate it

### Strategy Notes
- **Collection scale:** At ~$7k for half of one binder, total scanned value could reach $25k+ once commander decks and Binder 4 are included. Size of the haul should drive how carefully you sort and grade.
- **Commander decks:** Scan each deck as its own collection in Manabox (or tag by deck name) so you can evaluate per-deck sell value vs. keeping decks intact.
- **Bulk:** CK buylist is weak on bulk commons/uncommons. True bulk is better sold as a lot locally (Facebook Marketplace, LGS) or donated. Focus CK submission on singles $1+.
- **Condition matters:** NM = full buylist price; LP = ~70–80%. CK re-grades on arrival — be honest in your submission to avoid surprises.
- **Store credit vs. cash:** Decide this before Phase C submission — can't change after. Credit is the better play if reinvesting into sealed product or high-value singles.

## Tips for Success

1. **Protect mornings.** Baby-permitting, morning hours are often the highest-energy window. Start your hardest task by 8–9 AM.
2. **Phase-first mindset.** Don't think "I need to finish the stairs by Day 29." Think "I'm in the framing phase today." This keeps you moving even on slow days.
3. **Recruit help for Day 3.** A sleeper sofa without help is a back injury waiting to happen.
4. **Keep a running cut list.** Before every materials run, write out every piece by dimension so you only make one trip.
5. **Let paint cure.** Don't load the under-stair storage the same day you paint — give it 24 hours minimum.
6. **Don't over-engineer.** The IKEA Holmerud is a win precisely because it doesn't require building. Same for hairpin legs on the console table vs. milling your own.
7. **Document as you go.** Quick photos at each phase = before/after satisfaction + handy reference if something needs repairs later.
