# Context Budgeting
Always optimize token usage. Match the model tier to the task so context and cost stay lean.

## Model instructions
- **Opus** — architecting and high-level plans; tasks at this level of model.
- **Sonnet** — enacting code changes and appropriate tasks at this level of model.
- **Haiku** — quick searches, data gathering, and appropriate tasks at this level of model.
